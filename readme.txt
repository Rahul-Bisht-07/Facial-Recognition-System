# 🧠 Face Recognition System

A real-time face recognition system using OpenCV and `face_recognition`. It detects and recognizes faces from webcam input using a preloaded dataset of known images.

## ✨ Features
- Real-time webcam face detection and recognition
- Labeling of recognized individuals
- Simple image-based dataset (`/data`) structure
- Resize optimization for better performance

## 🛠️ Tech Stack
- Python
- OpenCV
- face_recognition (built on Dlib)
- NumPy
